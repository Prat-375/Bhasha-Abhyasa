# 🧠 German Learning App (Sanskrit Inspired)

A React-based web app to help users learn and practice German based on their level (A1–C1).

## 🚀 Features
- Select language level (A1–C1)
- Learn mode (structured topics)
- Practice mode:
  - Multiple choice
  - Fill in the blanks
  - Writing tasks

## 🛠 Tech Stack
- React
- JavaScript
- CSS

## 📦 Setup

```bash
npm install
npm run dev
